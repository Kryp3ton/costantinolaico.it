Ngl-Spammer

Ngl-Spammer è un tool open source che è stato creato sfruttando la vunerabilità trovata nel codice sorgente di "Ngl", essa permette tramite l'utilizzo di pochi proxy la possibilità di effettuare uno spam sia di grandi che di piccole dimensioni a un qualsiasi utente. Questo tool garantisce la sicurezza e l'incolumità dell'attaccante.
